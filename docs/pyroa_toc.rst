=======
PyROA
=======

PyROA is a recent tool used to correlate two light curves and find the time lag between the two using the running optimal average (ROA) technique.
Compared to the previous modules, PyROA is more complex and contains more nuances in its arguments. These tutorials provide instructions
on in-depth usage of PyROA and understanding of its arguments.
